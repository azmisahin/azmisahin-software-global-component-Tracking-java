# Tracking

The code architecture allows a simple method to be monitored.

# How it was done?

* JDK Install
* Maven Install
* Maven Generate
```cmd
mvn -B archetype:generate -DarchetypeGroupId=org.apache.maven.archetypes -DgroupId=com.azmisahin -DartifactId=tracking
```
